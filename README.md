### What is this

Its a blog with a cms were the user can add, update and delete posts. It also has an email sign up box with email authentication.

### Why was it built

Built to practice with node js. As I had built a blog before I had an idea of the structure of a blog so I chose this so I could focus on learning about node js and mongodb.

### Main technologies used
Javascript, Node.js, Mongodb, Nodemailer
